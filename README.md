# microblog
Advanced microblog app with Flask

## Prerequisites
* Python 3.7.0
* Flask 0.12.4
* flask-sqlalchemy
* flask-migrate
* flask-login

## Getting Started
### Linux
```python
cd microblog
export FLASK_APP=microblog.py
flask run
```

### Windows
```python
cd microblog
set FLASK_APP=microblog.py
flask run
```
